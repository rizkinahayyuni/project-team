## Studi kasus

2. Budi : A user in more than 1 project in a day (Project A 4 hours, Project B 4 hours)
3. Cinta : A user in more than 1 projects in a month (Project A 8 hours, Project B 8 hours)
4. Devi : How is the user absence in a day
